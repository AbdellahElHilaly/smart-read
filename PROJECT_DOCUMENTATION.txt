PROJECT SUMMARY
===============
Smart Read is a modern, interactive web application designed to help English 
learners practice reading comprehension by clicking words to reveal Arabic 
translations instantly. The application uses a clean, minimalist interface 
with smooth interactions and responsive design.

DATA FLOW DIAGRAM
=================
1. User opens index.html
   ↓
2. JavaScript loads data.json via fetch()
   ↓
3. createTranslationMap() builds lookup table
   ↓
4. renderText() creates span elements for each word
   ↓
5. attachEventListeners() adds click handlers
   ↓
6. User interacts with spans
   ↓
7. toggleTranslation() updates display and CSS classes
   ↓
8. CSS applies visual styles

SPAN-BASED ARCHITECTURE
=======================
Why Spans?
- Enable individual word targeting
- Support click event handling
- Allow independent styling
- Track translation state per word
- Provide clean data-binding approach

Implementation:
┌─ Original Text ─────────────────────────────┐
│ "Black holes are among the most fascinating" │
└──────────────────────────────────────────────┘
                    ↓
      ┌─ Split into Words ──┐
      │ ["Black", "holes", ...] │
      └─────────────────────┘
                    ↓
      ┌─ Create Spans for Each Word ─┐
      │ <span data-en="Black"         │
      │        data-ar="أسود">        │
      │   Black                       │
      │ </span>                       │
      └───────────────────────────────┘
                    ↓
      ┌─ Attach Event Listeners ─┐
      │ Click → Toggle Display   │
      │ Show Animation           │
      │ Update CSS Classes       │
      └──────────────────────────┘

TRANSLATION TOGGLE SYSTEM
==========================
State 1: English Display
- Text shows original English word
- CSS class: (none)
- Background: Normal
- Color: Default dark gray

State 2: Arabic Display (after click)
- Text shows Arabic translation
- CSS class: "translated"
- Background: Bright red (#e74c3c)
- Color: White text
- Font-weight: 600 (bold)

Toggling:
User Click ─→ Check Current State ─→ Switch Language ─→ Update CSS Class ─→ Play Animation

COLOR SCHEME
============
Primary: #2c3e50 (Dark Gray-Blue) - Main text
Secondary: #3498db (Blue) - Buttons, hover effects
Accent: #e74c3c (Red) - Translated words
Background: Linear gradient (Purple to lighter purple)
Light: #ecf0f1 (Light Gray) - Backgrounds

================================================================================
                           HOW TO USE
================================================================================

INSTALLATION
=============
1. Extract all files to a folder
2. Ensure all files are in the same directory:
   - index.html
   - style.css
   - script.js
   - data.json

LAUNCH APPLICATION
===================
Method 1: Direct Open
- Right-click index.html → Open with Browser
- Or drag into browser window

Method 2: Local Server (Recommended)
- Python: python -m http.server 8000
- Node.js: npx http-server
- VS Code: Live Server extension
- Then visit: http://localhost:8000

USING THE APPLICATION
======================
1. Read the displayed English text
2. Click on any word to reveal Arabic translation
3. Click again to toggle back to English
4. Use buttons:
   - "Reset Text": Returns all to English
   - "Clear All Translations": Removes active translations

================================================================================
                        CUSTOMIZATION GUIDE
================================================================================

ADD NEW WORDS
=============
File: data.json
Steps:
1. Open data.json in text editor
2. Add new objects:
   {
     "en": "new_english_word",
     "ar": "الكلمة الجديدة"
   }
3. Save file
4. Refresh browser - new words appear automatically

CHANGE COLORS
==============
File: style.css
Location: :root { ... } at top of file

Variables:
--primary-color: #2c3e50;      (Main text)
--secondary-color: #3498db;    (Buttons)
--accent-color: #e74c3c;       (Translated words)
--text-color: #2c3e50;         (Body text)
--light-bg: #ecf0f1;           (Light backgrounds)

Example Change:
Before: --accent-color: #e74c3c;
After:  --accent-color: #27ae60;  (Green instead of red)

CHANGE FONT SIZE
=================
File: style.css

Main text size: .text-content { font-size: 1.6em; }
Button text: .btn { font-size: 1em; }
Header: header h1 { font-size: 2.5em; }

MODIFY TEXT CONTENT TYPE
=========================
Option 1: Simple Update
- Keep data.json structure
- Update English and Arabic values only
- Perfect for different topics/difficulty levels

Option 2: Advanced Features
- Add more fields: pronunciation, difficulty, category
- Update JavaScript to display new fields
- Example: { "en": "word", "ar": "كلمة", "difficulty": 3 }

================================================================================
                           TROUBLESHOOTING
================================================================================

ISSUE: Words not displaying
SOLUTION:
- Check data.json exists in same folder
- Verify JSON syntax (use JSON validator)
- Check browser console for errors (F12)
- Ensure file is properly formatted

ISSUE: Clicking words does nothing
SOLUTION:
- Check script.js is loaded in browser console
- Verify addEventListener is working
- Ensure event delegation is correct
- Check CSS not blocking pointer events

ISSUE: Styling looks broken
SOLUTION:
- Refresh browser (Ctrl+Shift+R for hard refresh)
- Clear browser cache
- Check style.css file exists
- Verify no CSS syntax errors

ISSUE: Arabic text not showing correctly
SOLUTION:
- Check UTF-8 encoding in files
- Verify Arabic characters are valid
- Check HTML meta charset="UTF-8"
- Test in different browsers
